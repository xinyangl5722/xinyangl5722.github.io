CYOA Interactive Project

Context

During this project, I made an adventure project. I had work with a partner for this project. The project has many links to go from a file to another file. The title of this project is "CYOA-Travelling-To-Places"

Content

So the project starts with the player waking up in a random field and they have to choose which place to go to. Once the player goes into that chosen place, they have to continue with their adventure and see which object awaits them.

Reflection

Overall, this project helped me with my communication skills, allowing me to learn how to communicate even in long distances.

One of the challenges were to put the different files together with links. There were a lot of files to put together, so it was a bit difficult to put them all in a chain without any errors. I overcame this challenge by testing the chain of files to see where the error is and then fix the error and retesting it again.

If I could do this project differently, I would add more files into one file for both options. For example, instead for having two options in only one option, each option will have two more options behind them if you click either one of those options.
